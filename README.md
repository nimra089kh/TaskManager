# Task Manager System

A simple **Task Manager** application built using **ASP.NET Core MVC**.  
This application allows users to manage their tasks efficiently by creating, updating, and deleting tasks with a clean and user-friendly interface.

---

## Features

- **User Registration & Login:** Secure authentication for multiple users.  
- **Task CRUD Operations:** Users can **Create**, **Read**, **Update**, and **Delete** tasks.  
- **Task Status Management:** Mark tasks as pending, in-progress, or completed.  
- **Responsive UI:** Works well on both desktop and mobile devices.  
- **Clean MVC Structure:** Follows proper MVC architecture for easy maintainability.

---

## Technologies Used

- **ASP.NET Core MVC**  
- **C#** for backend logic  
- **Entity Framework Core** for database operations  
- **SQL Server** for storing user and task data  
- **Bootstrap** for UI styling
